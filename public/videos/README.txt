Place your hero sequence videos in this folder using these exact names:

1) mars-orbit-intro.mp4
   - Intro shot: arriving in Mars orbit and approaching station

2) station-interior-load.mp4
   - Interior transition where the site appears to load in layers

3) ambient-background-loop.mp4
   - Ambient loop that runs continuously behind the website interface

Notes:
- MP4 (H.264) is recommended for broad browser support.
- Keep each file compressed for web delivery (ideally <= 10-20 MB each where possible).
- If files are missing, the hero automatically falls back to /hero-mars-v2.jpg.
